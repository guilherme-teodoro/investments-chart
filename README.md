# investments chart



## O que foi usado

### Aplicação
- `create-react-app`
- `react` 
- `react-dom`
- `date-fns`
- `recharts`

### Testes
- `testing-library`
- `jest`


## Começando
```
# Instalar dependencias
$ yarn

# Subindo a aplicação
$ yarn start
```
Abrir no browser em `http://localhost:3000` 

## Testes
```
$ yarn test
```

## Buildando
```
$ yarn build
```
